<h1 align="center">Lunvia AI Journaling App</h1>

<hr style="border:1px solid #eee; margin: 20px 0;" />

<h2>Overview</h2>
<p>
Lunvia is a React Native + TypeScript mobile application designed for <strong>AI-assisted journaling</strong>. 
Users can write daily entries, track mood patterns, and receive intelligent insights powered by AI.  
Built as part of my personal journey to learn React Native, this project focuses on habit tracking, reflection, and AI guidance while exploring cross-platform mobile development.
</p>

<hr style="border:1px solid #eee; margin: 20px 0;" />

<h2>Key Features</h2>
<ul>
<li>Write daily journal entries with rich prompts</li>
<li>AI-powered chat assistant to reflect on entries, suggest prompts, and provide insights</li>
<li>Mood tracking and visualization of trends over time</li>
<li>Streaks and milestone tracking to encourage consistent journaling</li>
<li>Premium features including personalized advice, unlimited entries, and theme customization via Clerk authentication and billing</li>
<li>Cross-platform support (iOS & Android)</li>
<li>Clean code structure with modular components and TypeScript type safety</li>
</ul>

<hr style="border:1px solid #eee; margin: 20px 0;" />

<h2>Tech Stack & Libraries</h2>
<ul>
<li>React Native (TypeScript)</li>
<li>UI & Styling: Tamagui for components and layout</li>
<li>Authentication & Billing: Clerk</li>
<li>AI Integration: Custom chat assistant service, OpenAI API wrappers</li>
<li>Navigation: React Navigation</li>
<li>Project configuration: Babel, Metro, tsconfig — for type support & React Native build flow</li>
</ul>

<h2>How to Run the App</h2>
<ol>
<li>Download or clone the project repository</li>
<li>Install dependencies with <code>npm install</code> or <code>yarn install</code></li>
<li>Add your AI API key (OpenAI or other) in a <code>.env</code> file</li>
<li>Start the app with <code>npx expo start</code> or <code>react-native start</code></li>
<li>Run on iOS simulator, Android emulator, or Expo Go</li>
</ol>
